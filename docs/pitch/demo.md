# Demo Flow

## Live Monitor Segment

1. Open the dashboard on the monitor tab.
2. Show timeline range changes and replay cursor controls.
3. Highlight incident queue, evidence feed, and geo map.
4. Open the triage drawer on an incident or event.

## Supply Chain Segment

1. Go to Settings and connect GitHub.
2. Select a repository.
3. Start a fast or full scan.
4. Show scan progress, live activity, and ADK pipeline trace.
5. Review dependency inventory, vulnerability list, and code findings.

## Settings Segment

1. Save a Gemini API key.
2. Switch the Gemini model.
3. Configure GCP Cloud Logging.

## Close

- Return to the unified value proposition:
  - runtime visibility
  - repository visibility
  - AI-assisted triage in one product
